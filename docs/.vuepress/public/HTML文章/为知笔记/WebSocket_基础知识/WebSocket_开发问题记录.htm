遇到问题：new WebSocket 对象时报错：ReferenceError: WebSocket is not defined


问题描述：
    在 Node.js 中 new WebSocket 对象，准备与后端建立 WebSocket 长连接时，报错：
        ReferenceError: WebSocket is not defined
        引用错误：WebSocket 没有被定义
问题原因：
网查资料，在 stackoverflow 上找到了答案：
当您尝试运行 JavaScript 时，通常会在浏览器或 NodeJS 中运行它。但是，浏览器和 NodeJS 是不同的 JavaScript 运行时。NodeJS 不提供任何 API 供开发人员使用 WebSocket，但是浏览器提供。WebSocket 是 H5 特性，只要支持 H5 的浏览器就支持 WebSocket API 。
WebSocket 本身是一个应用层协议，你需要使用 API 来使用它。
那么如何在 NodeJS 上实现 WebSockets（客户端）？需要第三方库，如 https://github.com/websockets/ws 。

问题解法：
    问题解法有二：1、在支持 H5 的浏览器中运行；2、引用第三方库。

Ref：
    stackoverflow - JavaScript - ReferenceError: WebSocket is not defined
    https://stackoverflow.com/questions/43311238/javascript-referenceerror-websocket-is-not-defined







遇到问题：启动 WebSocketServer 时报错： SyntaxError: Unexpected token {


问题描述：
    在 Node.js 中启动 WebSocketServer 时报错： 
        SyntaxError: Unexpected token {
        语法错误：未预料的符号“{”

问题原因：
    V8引擎不支持 export、import 这样语法的导入导出 ，详见 https://segmentfault.com/q/1010000010921087

问题解法：
    安装 babel
