JavaScript 文件上传、下载



Q：Java 和 JS 的文件上传和下载有什么区别？

A：Java 是自身的功能，JS 没有这个功能。
为什么？因为 Java 有自己的 Runtime，如 JVM，JS 有自己的 Runtime 吗？没有，JS 必须依托于浏览器的 Runtime 或者其他第三方 Runtime（如 Node.js）。所以说，JS 那些文件上传下载的功能都是浏览器的功能，是浏览器实现了这些 Web API，与 JS 本身无瓜。但这不能怪 JS，毕竟 JS 是以 Script 血统诞生的。

这也就是为什么在下载文件时，要让启动下载窗口的话， JS 需要借助  <a> 才能让浏览器启动下载窗口，而 Java 只需要靠自己就能启动自己的下载窗口。
